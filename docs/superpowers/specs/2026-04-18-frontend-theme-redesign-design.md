# KapGame 前端主题重设计规范

## 概述

将 KapGame 微信小程序前端从 Cinema Glass 深色主题重新设计为**柔和玻璃双主题**，支持**系统级自动切换**白天/黑夜模式。

## 设计风格

**柔和玻璃风格 (Soft Glass)** — 轻量毛玻璃效果，淡雅渐变，柔和阴影，适合阅读的浅色主题。

## 核心设计语言

### 玻璃效果
- `backdrop-filter: blur(12px)` — 轻量毛玻璃
- 背景不透明度：白天 70%，黑夜 60%
- 边框：`1px solid` 细线

### 渐变
- 角度：`145deg` 柔和渐变
- 色彩过渡自然不刺眼
- 模块使用实色渐变而非半透明

### 阴影
- 白天模式：`0 4px 24px rgba(0,0,0,0.06)`
- 黑夜模式：`0 4px 24px rgba(0,0,0,0.3)`
- 模块独立阴影增强立体感

### 圆角
- `20rpx` 中等圆角
- 柔和但有性格

### 字体
- 纯色不透明，保证可读性
- 标题加 `font-weight: 700`

### 3D 效果
- 顶部细线高光表现光泽
- 底部淡阴影表现层次
- 不使用半透明字体

## 实现方式

### 主题检测
利用微信小程序系统主题检测，通过 `page` 伪类选择器自动切换。

```css
/* 默认明亮模式 */
page {
  --bg-primary: #FAFBFC;
  /* ... 其他变量 */
}

/* 深色模式 */
@media (prefers-color-scheme: dark) {
  page {
    --bg-primary: #0F1419;
    /* ... 覆盖为深色变量 */
  }
}
```

### 页面覆盖范围
1. `app.wxss` — 全局变量和通用组件
2. `pages/shop/shop.wxss` — 商城页
3. `pages/home/home.wxss` — 对战主页
4. `pages/chat/chat.wxss` — 消息页
5. `pages/login/login.wxss` — 登录页
6. `components/userCard/userCard.wxss` — 用户卡片组件
