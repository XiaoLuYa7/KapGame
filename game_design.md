# 🎮 游戏设计方案：《时序裂隙（Temporal Rift）》

> 主题：未来科幻 + 时间错乱
>  核心体验：操控时间线、规避“时间崩塌”，让别人踩坑

------

## 一、核心玩法（重构版）

- 每回合：
  1. 可使用任意张手牌（不限1张）
  2. **必须进行一次“时间抽取”（抽牌）**
- 抽到【时间崩塌】：
  - 若无【稳定锚】，直接淘汰
  - 若有，则可将该卡插入牌堆任意位置

✔ 本质仍是“抽炸弹”，但包装为时间系统
 ✔ 加强“信息操控”和“顺序干预”

------

## 二、模式设计

### 👤 单人模式（5人局）

- 5人混战（含AI或真人）
- 崩塌数量：**玩家数 - 1 = 4张**
- 最后存活 → 第一名
- 奖励：金币（按排名梯度）

------

### 👥 组队模式（6人局 / 3v3）

- 3个队伍（每队2人）
- 任意一队全灭 → 出局
- 最后存活队伍获胜
- 奖励：队伍共享金币

✔ 队友之间：

- **不能直接看手牌**
- 但可通过部分卡牌“间接协作”

------

## 三、牌堆结构设计

总牌数：

- 单人模式：53张
- 组队模式：60张（增加干扰与协作）

------

## 四、卡牌系统设计（核心）

### 💥 1. 核心死亡机制

| 卡牌名   | 数量      | 功能                 |
| -------- | --------- | -------------------- |
| 时间崩塌 | 单4 / 双5 | 抽到若无防御 → 淘汰  |
| 稳定锚   | 单6 / 双7 | 抵消崩塌，并重新插入 |

------

| 卡牌名   | 数量      | 功能                |
| -------- | --------- | ------------------- |
| 时间观测 | 单4 / 双5 | 查看牌堆顶部3张     |

| 未来回溯 | 单3 / 双4 | 偷看任意玩家手牌1张 |

------

### 🔄 3. 操控类（强策略）

| -------- | --------- | ------------------ |

| 量子窃取 | 单4 / 双4 | 向玩家要1张牌      |
| 时序重排 | 单4 / 双5 | 洗牌               |

------

### ⏭️ 4. 节奏控制类

| 卡牌名   | 数量      | 功能           |
| -------- | --------- | -------------- |

| 逆向流动 | 单5 / 双6 | 跳过并反转顺序 |
| 因果转移 | 单5 / 双6 | 让别人替你抽   |
| 连锁转移 | 单3 / 双3 | 让目标抽2次    |

------

### 🎯 5. 风险操作类（新增亮点）

| 卡牌名   | 数量      | 功能                              |
| -------- | --------- | --------------------------------- |
| 裂隙探测 | 单4 / 双4 | 从牌堆底摸                        |
| 时间锁定 | 单2 / 双3 | 指定一张牌位置不可改变            |
| 崩塌标记 | 单2 / 双3 | 标记某人，下次抽牌+1风险（抽2张） |

------

## 五、核心创新点（重点）

### 1️⃣ “时间顺序”可被操控

不是简单洗牌，而是：

- 插入位置
- 锁定位置
- 预测位置

👉 增强“算牌深度”

------

### 2️⃣ 组队模式的信息不完全共享

- 不能直接看队友牌
- 需要通过行为判断

👉 增强“社交推理”

------

### 3️⃣ 风险叠加机制（新）

- 抽2张、连续抽
- 风险可“转嫁”

👉 比原版更刺激

------

## 六、经济系统（简化）

### 💰 金币规则

| 排名 | 奖励 |
| ---- | ---- |
| 第1  | 100  |
| 第2  | 60   |
| 第3  | 30   |
| 第4  | 10   |
| 第5  | 0    |

组队模式：

- 胜利队伍：每人 100
- 失败队伍按存活时间给

------

## 七、整体体验定位

| 维度 | 设计               |
| ---- | ------------------ |
| 节奏 | 快（5~10分钟一局） |
| 上手 | 简单               |
| 深度 | 中等（博弈+心理）  |
| 风格 | 轻松 + 科幻        |
| 情绪 | 紧张 → 爆笑        |

------

## 八、可扩展方向（给你后面做版本用）

### 🔥 后续版本可以加：

- “技能角色”（每人被动能力）
- “事件卡”（全局效果）
- “特殊胜利条件”
- 排位系统 / 段位

------

## 九、总结（重点）

你原本那套：
 👉 偏“纯运气+简单策略”

现在这套：
 👉 变成 **运气 + 信息 + 博弈 + 队友协作**

但仍然：

- 规则简单（可小程序落地）
- 卡牌数量可控
- 非重度游戏（符合休闲定位）
